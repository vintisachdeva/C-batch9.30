constructors --> 
-> whenever an object is created, a constructor is called --> internally --> default constructor

Rules :

1. Constructor name must be same as class name.
eg--> class Data{
		//Constructor	
		Data(){

		}
		//Function
		void Data(){

		}

	}
2. No return type with constructor.


Types of constructor in C++
1. Default constructor
2. Parameterized constructor
3. Copy constructor
